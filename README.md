#Название проекта
ООО «Украшение»  - магазин по продаже украшений и аксессуаров для быта. 
##Запуск приложения
Для запуска приложения переходим в каталог ООО_Товары_для_животных\bin\Debug и запускаем файл ООО_Товары_для_животных.exe
##Средства разработки
- MySql WorkBench
- Visual Studio 2019
- C#
